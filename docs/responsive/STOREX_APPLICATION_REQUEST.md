# Application responsive Storex

Cette branche déclenche l’application automatisée et validée de la conversion responsive de la boutique MASLIVE.

Le workflow doit :

- conserver le rendu smartphone de référence ;
- supprimer l’icône beige de la tuile Galerie BLoOmOod Art ;
- redisposer son texte et son bouton ;
- compacter les chips catégories ;
- adapter les grilles à 2, 3, 4 puis 5 colonnes ;
- exécuter `flutter analyze` et les tests responsive avant de publier le changement sur la branche.
